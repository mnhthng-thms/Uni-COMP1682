How is This Page Rendered?

When a request is sent to the HTTP endpoint of / (default endpoint), HomeController of this server application will render the return HTML document based on template file index.html

Bootstrap Database Connection

Either mysql -u root -p or using MySQL Workbench 

    /* 0 */
    SELECT * FROM mysql.user;
    /* 1 */
    CREATE USER 'temp'@'localhost' IDENTIFIED BY 'holaholaheh';
    /* 2 */
    UPDATE `mysql`.`user`
    SET `Select_priv` = 'Y', `Insert_priv` = 'Y', `Update_priv` = 'Y', `Delete_priv` = 'Y'
    WHERE (`Host` = 'localhost') and (`User` = 'temp');

In application.properties of this server application project: 

    server.port=9090

    spring.datasource.url=jdbc:mysql://localhost:3306/
    spring.datasource.username=temp
    spring.datasource.password=holaholaheh
    spring.datasource.driver-class-name=com.mysql.cj.jdbc.Driver
    spring.jpa.database-platform=org.hibernate.dialect.MySQL5Dialect
    spring.devtools.livereload.enabled=true
